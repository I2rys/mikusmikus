# MikusMikus
Annoying malware that will move all your files in random locations.

## Installation
Github:
```
git clone https://github.com/I2rys/mikusmikus
```

NpmJS:
```
npm i path os fs
```

## Build
Use [pkg](https://www.npmjs.com/package/pkg) to build MikusMikus and make sure to compress it.

## Usage
```
node index.js
```
